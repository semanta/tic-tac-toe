# Intro to react 
An interactive tic-tac-toe game


- Create React App

  Create React apps with no build configuration.

    `> npm install -g create-react-app`
    
- install modules

   `> npm install`

- start app
  
  `> npm start`
  
   Runs the app in development mode. Open http://localhost:3000 to view it in the browser.
